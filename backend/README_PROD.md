SQL Copilot Admin is an AI-powered database management tool designed for efficiency and simplicity across multiple database engines.

## Key Features

1. **AI SQL Copilot**: Write queries in natural language (NL-to-SQL).
2. **Smart Optimizer**: Analyze execution plans and get optimization suggestions.
3. **Visual Database Explorer**: Interactive ER diagrams and dependency maps.
4. **Excel Importer**: High-speed, staged data migration with Upsert and Auto-Sync support.
5. **Multi-Engine Support**: Firebird, PostgreSQL, MySQL, SQL Server, and SQLite.
6. **Bulk Tools**: Mass Find & Replace and Fetch All capabilities.
7. **User Management**: Database user and password administration.
